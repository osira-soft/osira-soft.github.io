収集する情報

TideScope は、いかなる個人情報も収集しません。

アプリが保存するデータは以下のみです。これらはすべてお使いのデバイス上にのみ保存され、外部に送信されることはありません。

  • お気に入り登録した観測地点
  • 最後に選択した観測地点
  • アプリ内の設定情報

位置情報

TideScope は、最寄りの観測地点を自動選択するために位置情報を使用することがあります。位置情報はデバイス内でのみ処理され、外部サーバーに送信されることはありません。

外部 API の利用

潮位データの取得に気象庁の公開 API を、天気データの取得に Open-Meteo（オープンソース気象 API）を使用しています。これらへのリクエストには個人を特定できる情報は含まれません。

サードパーティへの情報提供

いかなる情報も第三者に提供・販売・共有することはありません。
